## ⚡ شروع فوری (3 دقیقه)

### گام 1️⃣: نصب وابستگی‌ها
```bash
pip install -r requirements.txt
```

### گام 2️⃣: تنظیم فایل config.py
```bash
# اگر فایل config.py وجود ندارد:
cp config.example.py config.py
```

سپس این 3 خط را در `config.py` ویرایش کنید:

```python
TOKEN = "توکن_ربات_تلگرام_شما"           # از @BotFather
CHANNEL_ID = "@نام_کانال_شما"            # یا -1001234567890
GEMINI_API_KEYS = [
    "کلید_API_جمنای_اول",                # از aistudio.google.com
    "کلید_API_جمنای_دوم",                # اختیاری اما توصیه می‌شود
]
```

### گام 3️⃣: اجرای ربات
```bash
python bott.py
```

## 🔑 کجا کلیدها را بگیریم؟

### 🤖 توکن ربات تلگرام:
1. در تلگرام به [@BotFather](https://t.me/botfather) بروید
2. `/newbot` بزنید
3. دستورات را دنبال کنید
4. توکن را کپی کنید

### 📢 آیدی کانال:
**روش 1:** اگر کانال یوزرنیم عمومی دارد:
```python
CHANNEL_ID = "@your_channel"
```

**روش 2:** استفاده از آیدی عددی:
1. [@userinfobot](https://t.me/userinfobot) را به کانال اضافه کنید
2. آیدی را کپی کنید (مثل: `-1001234567890`)

**⚠️ مهم:** ربات را حتماً **ادمین** کانال کنید!

### 🧠 کلید Gemini AI:
1. به [Google AI Studio](https://aistudio.google.com/apikey) بروید
2. با اکانت گوگل لاگین کنید
3. "Create API Key" بزنید
4. کلید را کپی کنید

**💡 نکته:** برای پایداری بیشتر، 2-3 کلید بسازید
