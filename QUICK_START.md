# 🚀 راهنمای سریع راه‌اندازی

<div dir="rtl">

## گام‌های راه‌اندازی در 5 دقیقه

### 1️⃣ نصب کتابخانه‌ها

```bash
pip install -r requirements.txt
```

### 2️⃣ دریافت توکن ربات تلگرام

1. به [@BotFather](https://t.me/BotFather) در تلگرام پیام دهید
2. دستور `/newbot` را ارسال کنید
3. نام و username برای ربات انتخاب کنید
4. توکن دریافتی را کپی کنید

### 3️⃣ ایجاد کانال تلگرام

1. یک کانال عمومی یا خصوصی در تلگرام بسازید
2. ربات را به عنوان ادمین به کانال اضافه کنید
3. username کانال (مثل `@military_flights`) یا ID عددی آن را یادداشت کنید

**نکته**: برای دریافت ID عددی کانال، از [@userinfobot](https://t.me/userinfobot) استفاده کنید.

### 4️⃣ دریافت کلید API جمنای

1. به [Google AI Studio](https://aistudio.google.com/app/apikey) بروید
2. با اکانت گوگل خود وارد شوید
3. روی "Create API Key" کلیک کنید
4. کلید دریافتی را کپی کنید

**نکته مهم**: می‌توانید چندین کلید API ایجاد کنید تا در صورت محدود شدن، ربات به صورت خودکار سوییچ کند.

### 5️⃣ ویرایش فایل پیکربندی

فایل `config.py` را با ویرایشگر متن باز کنید و تغییرات زیر را اعمال کنید:

```python
# توکن ربات تلگرام (از BotFather دریافت کرده‌اید)
TELEGRAM_BOT_TOKEN = "1234567890:ABCdefGHIjklMNOpqrsTUVwxyz"

# آیدی کانال تلگرام
TELEGRAM_CHANNEL_ID = "@military_flights"  # یا "-1001234567890"

# آیدی کاربری تلگرام شما برای دریافت گزارش‌های روزانه
ADMIN_USER_ID = 123456789  # آیدی عددی خودتان

# کلیدهای API جمنای (می‌توانید 3-5 کلید اضافه کنید)
GEMINI_API_KEYS = [
    "AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX",  # کلید اول
    "AIzaSyYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYY",  # کلید دوم (اختیاری)
    "AIzaSyZZZZZZZZZZZZZZZZZZZZZZZZZZZZZZZZ",  # کلید سوم (اختیاری)
]
```

**نکته**: برای دریافت آیدی عددی خودتان، به [@userinfobot](https://t.me/userinfobot) پیام دهید.

### 6️⃣ اجرای ربات

```bash
python3 bot.py
```

اگر همه چیز درست باشد، خروجی زیر را خواهید دید:

## ✅ تست ربات

### تست 1: ارسال پیام شروع

1. به ربات خود در تلگرام پیام دهید
2. دستور `/start` را ارسال کنید
3. باید پیام خوش‌آمدگویی دریافت کنید

### تست 2: بررسی وضعیت

دستور `/status` را ارسال کنید. باید آمار فعلی ربات را ببینید.

### تست 3: ارسال پیام تست به کانال

دستور `/test` را ارسال کنید. باید یک پیام تست در کانال ظاهر شود.

### تست 4: بررسی AI

دستور `/ai` را ارسال کنید تا وضعیت سیستم هوش مصنوعی را ببینید.

## 🎯 نکات مهم

### ✅ اگر همه چیز کار می‌کند:

- هر 2 دقیقه یکبار، ربات پروازهای نظامی را بررسی می‌کند
- پروازهای جدید با تحلیل AI به کانال ارسال می‌شوند
- هر ساعت یک گزارش آماری به کانال ارسال می‌شود
- هر 24 ساعت یک گزارش از هواپیماهای ناشناس به شما ارسال می‌شود

### ❌ اگر مشکلی وجود دارد:

1. **خطای "invalid token"**
   - توکن ربات را دوباره بررسی کنید
   - مطمئن شوید که بدون فاصله کپی شده است

2. **خطای "chat not found"**
   - ربات را به عنوان ادمین به کانال اضافه کنید
   - مطمئن شوید ID کانال صحیح است

3. **خطای "API key invalid"**
   - کلید Gemini را در [AI Studio](https://aistudio.google.com/app/apikey) بررسی کنید
   - مطمئن شوید که کلید فعال است

4. **ربات اجرا می‌شود اما پروازی نمی‌فرستد**
   - صبر کنید، ممکن است چند دقیقه طول بکشد
   - `MIN_STRATEGIC_IMPORTANCE` را در config به 3 کاهش دهید
   - لاگ فایل `military_tracker.log` را بررسی کنید

## 📊 تنظیمات پیشنهادی

### برای استفاده شخصی (کم‌هزینه)

```python
POLL_INTERVAL_SECONDS = 300        # هر 5 دقیقه
MIN_STRATEGIC_IMPORTANCE = 7       # فقط مهم‌ترین‌ها
USE_AI_CACHE = True                # حتماً فعال
```

### برای کانال عمومی (پر‌محتوا)

```python
POLL_INTERVAL_SECONDS = 120        # هر 2 دقیقه
MIN_STRATEGIC_IMPORTANCE = 5       # همه پروازها
USE_AI_CACHE = True                # حتماً فعال
```

### برای مانیتورینگ منطقه خاص

در `config.py` → `STRATEGIC_REGIONS` فقط مناطق مورد نظر را نگه دارید:

```python
STRATEGIC_REGIONS = {
    "خلیج فارس": (24.0, 30.5, 48.0, 57.0),
    "خاورمیانه": (12.0, 42.0, 34.0, 63.0),
}
```
