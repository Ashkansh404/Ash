# 🚀 ربات هوشمند ردیاب پروازهای نظامی
## Intelligent Military Flight Tracker Bot v15.0

یک ربات تلگرام پیشرفته برای ردیابی و تحلیل هوشمند پروازهای نظامی با استفاده از هوش مصنوعی Gemini 2.5 Pro

---

## ✨ قابلیت‌های پیشرفته

### 🤖 تحلیل هوشمند با AI
- **ادغام با Gemini 2.5 Pro**: تحلیل عمیق هر پرواز با استفاده از هوش مصنوعی
- **سوییچ خودکار API Key**: چرخش خودکار بین چندین کلید API در صورت محدودیت
- **پرامپت مهندسی شده**: دستورات بهینه‌شده برای بهترین نتایج تحلیلی

### 💾 سیستم یادگیری و کش هوشمند
- **یادگیری مداوم**: ذخیره تحلیل‌های AI در پایگاه داده محلی
- **پایگاه داده افلاین رو به رشد**: هر پرواز تحلیل شده به دانش ربات اضافه می‌شود
- **جستجوی چندلایه**: جستجو بر اساس ICAO، Callsign، و الگوهای تاریخی
- **امتیاز اعتماد (Confidence Score)**: ارزیابی کیفیت هر تحلیل

### 🛡️ فالبک چندلایه (Zero-Downtime)
1. **کش دیتابیس** (بر اساس ICAO)
2. **جستجوی Callsign** (برای هواپیماهای شناخته شده)
3. **تحلیل AI با Gemini** (تحلیل جدید)
4. **دیکشنری‌های محلی** (روش سنتی در صورت خرابی AI)

### 📊 تحلیل‌های پیشرفته
- شناسایی دقیق نقش و مدل هواپیما
- تحلیل موقعیت استراتژیک (فارسی، حداکثر 15 کلمه)
- تشخیص اپراتور نظامی
- امتیاز اهمیت استراتژیک (1-10)
- تحلیل منطقه جغرافیایی

### 📈 آمار و گزارش‌دهی
- گزارش ساعتی عملکرد
- گزارش روزانه پروازهای ناشناس
- آمار استفاده از API ها
- آمار پایگاه داده

## 📋 پیش‌نیازها

### نرم‌افزاری
- Python 3.8 یا بالاتر
- pip (مدیر بسته Python)

### API و کلیدها
1. **Telegram Bot Token**: از [@BotFather](https://t.me/botfather) دریافت کنید
2. **Telegram Channel ID**: آیدی کانال برای ارسال اعلان‌ها
3. **Gemini API Keys**: از [Google AI Studio](https://aistudio.google.com/apikey) دریافت کنید (حداقل 1، توصیه می‌شود 3 کلید)

## 🔧 نصب و راه‌اندازی

### مرحله 1: کلون یا دانلود پروژه
```bash
# اگر از git استفاده می‌کنید:
git clone <repository-url>
cd military-flight-tracker

# یا فایل‌ها را مستقیماً دانلود کنید
```

### مرحله 2: نصب وابستگی‌ها
```bash
pip install -r requirements.txt
```

### مرحله 3: تنظیم فایل config.py
فایل `config.py` را باز کرده و اطلاعات زیر را وارد کنید:

```python
# Telegram Configuration
TOKEN = "1234567890:ABCdefGHIjklMNOpqrsTUVwxyz"  # توکن ربات تلگرام
CHANNEL_ID = "@your_channel"  # یا -1001234567890
ADMIN_ID = 123456789  # آیدی عددی شما

# Gemini AI Configuration
GEMINI_API_KEYS = [
    "AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX",  # کلید اول
    "AIzaSyYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYY",  # کلید دوم
    "AIzaSyZZZZZZZZZZZZZZZZZZZZZZZZZZZZZZZZ",  # کلید سوم
]
```

**نکات مهم:**
- حداقل یک کلید Gemini الزامی است
- استفاده از چند کلید باعث می‌شود در صورت محدودیت، ربات متوقف نشود
- کلیدهای رایگان Gemini محدودیت 15 درخواست در دقیقه دارند

### مرحله 4: اجرای ربات
```bash
python bott.py
```

یا در حالت daemon (پس‌زمینه):
```bash
nohup python bott.py > bot_output.log 2>&1 &
```

## 🎮 دستورات ربات

ربات از دستورات زیر در تلگرام پشتیبانی می‌کند:

| دستور | توضیحات |
|-------|---------|
| `/start` | نمایش راهنما و معرفی ربات |
| `/status` | وضعیت فعلی ربات و آمار لحظه‌ای |
| `/stats` | آمار پایگاه داده (پروازهای یادگرفته شده) |
| `/apistats` | آمار استفاده از API های مختلف |
| `/test` | ارسال پیام تست به کانال |

## ⚙️ تنظیمات پیشرفته (config.py)

### پارامترهای قابل تنظیم:

```python
# بازه زمانی بررسی پروازها (ثانیه)
POLL_INTERVAL_SECONDS = 120  # پیشنهاد: 60-180

# مدل Gemini (انتخاب بر اساس نیاز)
GEMINI_MODEL = "gemini-2.0-flash-exp"  # سریع و رایگان
# یا: "gemini-1.5-pro"  # دقیق‌تر اما کندتر
# یا: "gemini-2.5-pro"  # پیشرفته‌ترین (در صورت دسترسی)

# تاخیر بین ارسال پیام‌ها (ثانیه)
MESSAGE_DELAY = 40  # جلوگیری از محدودیت تلگرام

# پاکسازی رکوردهای قدیمی (ساعت)
MEMORY_CLEANUP_HOURS = 24

# سطح لاگ
LOG_LEVEL = "INFO"  # DEBUG, INFO, WARNING, ERROR, CRITICAL
```

## 🔍 نحوه عملکرد سیستم یادگیری

### جریان تحلیل پرواز:

```
پرواز جدید شناسایی شد
    ↓
1. جستجو در کش (بر اساس ICAO)
    ↓ (اگر یافت نشد)
2. جستجو بر اساس Callsign
    ↓ (اگر یافت نشد)
3. ارسال به Gemini AI برای تحلیل
    ↓
4. ذخیره نتیجه در دیتابیس
    ↓ (اگر Gemini در دسترس نبود)
5. فالبک به دیکشنری‌های محلی
    ↓
ارسال اعلان تلگرام
```

### رشد پایگاه داده:
- هر پرواز جدید که توسط AI تحلیل می‌شود، به دیتابیس اضافه می‌شود
- دفعات بعدی که همان هواپیما رصد شود، از کش استفاده می‌شود (سریع‌تر و بدون مصرف API)
- پس از چند روز فعالیت، ربات بیش از 80% پروازها را بدون نیاز به AI شناسایی می‌کند

## 🛠️ عیب‌یابی

### ربات راه نمی‌افتد:
```bash
# بررسی نسخه Python
python --version  # باید 3.8+ باشد

# بررسی نصب وابستگی‌ها
pip list | grep telegram

# مشاهده لاگ‌های خطا
tail -f bot.log
```

### Gemini جواب نمی‌دهد:
- کلید API را در [Google AI Studio](https://aistudio.google.com) بررسی کنید
- محدودیت رایگان: 15 درخواست/دقیقه (به همین دلیل چند کلید استفاده کنید)
- مدل را به `gemini-2.0-flash-exp` تغییر دهید (سریع‌تر و پایدارتر)

### پیام‌ها ارسال نمی‌شوند:
- بررسی کنید ربات ادمین کانال است
- CHANNEL_ID را بررسی کنید (باید با @ یا منفی شروع شود)
- محدودیت تلگرام: 20 پیام/دقیقه در کانال‌های عمومی
